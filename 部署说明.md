# 部署说明

当前文件夹已经是可部署的静态网站，入口是 `index.html`。

## 推荐方式：GitHub Pages

前提：先在本机登录 GitHub CLI。

```powershell
gh auth login
```

登录完成后，在本目录执行：

```powershell
git init
git add .
git commit -m "发布视频作品集网站"
gh repo create 视频作品集网站 --public --source=. --remote=origin --push
gh api repos/:owner/视频作品集网站/pages -X POST -f source.branch=main -f source.path=/
```

部署成功后，网站地址通常是：

```text
https://你的GitHub用户名.github.io/视频作品集网站/
```

如果仓库创建时默认分支不是 `main`，把命令里的 `main` 改成实际分支名。

## 备选方式：Vercel

如果你更想用 Vercel，可以把整个 `视频作品集网站` 文件夹拖到 Vercel 项目里，Framework 选择 `Other` 或不选框架，Build Command 留空，Output Directory 留空。

## 备选方式：Netlify

把整个 `视频作品集网站` 文件夹拖到 Netlify Drop 页面即可，入口文件已经在根目录。

## 注意

- 不能只上传 `index.html`，必须上传整个文件夹，因为视频、封面、CSS 和 JS 都在子目录里。
- `DJI无声三界视角` 当前是进行中项目，页面中已经按进行中标记。
- 联系邮箱当前是 `hello@example.com`，正式发布前建议替换。
